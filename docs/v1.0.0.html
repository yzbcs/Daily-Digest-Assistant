🎉 首个可用版本

Arxiv Daily Digest 首个正式版本发布，核心功能全部可用。

✨ 核心功能

  • 📬 每日邮件推送：北京时间每天中午自动推送最新 arXiv 论文 + 小红书笔记
  • 🤖 LLM 智能筛选：自动从候选论文中筛选最相关的 Top N 篇
  • 📝 中文双层摘要：一句话总结 + 详细解读
  • 📕 小红书同步：每日关键词搜索 + LLM 筛选，不受 arXiv 休息日影响
  • 🔄 去重机制：记录已推送论文，不重复推送
  • ⚙️ 纯 GitHub Actions：fork 即用，无需服务器

🛠️ 技术栈

  • Python 3.11 + GitHub Actions
  • 多 LLM 支持：Claude / MiniMax / OpenAI / DeepSeek / 智谱 / Kimi / 通义
  • HTML 邮件（Jinja2 模板）
  • 小红书 API 逆向（Node.js 签名）
