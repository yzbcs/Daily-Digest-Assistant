✨ 自定义 LLM 提供商

最大的变化：新增 custom_llm 配置项，让你无需修改任何代码就能接入任意 LLM API。

适用场景：你有一个自建 API、第三方代理，或者任何不在内置列表中的模型服务。

在 config.yml 中这样配置：

llm_provider: my_api

custom_llm:
  my_api:
    sdk: openai
    base_url: "https://my-api.com/v1"
    model: "gpt-4o"

支持的 sdk 类型：

  • openai — OpenAI 兼容协议（绝大多数国产模型、自建服务）
  • anthropic — Anthropic 兼容协议（Claude、MiniMax 等）

🧹 清理非官方提供商

移除了以下非官方内置提供商：

  • xingjiabiapi — 第三方代理服务
  • opencode — 第三方代理服务

如果你之前在使用这些提供商，只需在 custom_llm 中按上面的方式重新配置即可，功能完全不受影响。
